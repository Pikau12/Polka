# Polka

Android app for board games collection and sessions tracking.

## Tech stack
### 📱 Android
- Kotlin
- Jetpack Compose + Material3
- MVVM
- Room (offline DB)
- Retrofit + OkHttp
- WorkManager (sync + background tasks)
- DataStore (settings)
- Coil (images)
- Hilt (DI)
- Repository pattern (offline-first logic)

### 🖥 Backend
- Go
- Gin (HTTP framework)
- PostgreSQL
- sqlx (DB access)
- JWT + Refresh tokens
- Modular monolith architecture
- BGG integration module
- Swagger/OpenAPI (API contract)
